Email Header Footer is a wordpress plugin that lets you cutomize email by setting header and footer of email.

== Description ==
Email Header Footer plugin allows you to set layout of mail. Email Header Footer sets header and footer of email which sent by native wp_mail function. Email Header Footer make easy to set logo in header and signature in footer of email. This plugin provide many short code which you can put in header and footer such as [site-title],  [tag-line],  [site-url],  [admin-email], [year].
== Installation ==
Upload the Email Header Footer plugin to your wordpress website, Activate it.
